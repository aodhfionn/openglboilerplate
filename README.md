# OpenGL Boilerplate

This is some boilerplate code for the beggining to almost any OpenGL Project

# Develop: Get Started

1. Install dependencies.
    - OpenGL
    - GLFW (X11)
    - GLAD (see [glad/README.md](glad/README.md))
    - FreeType
2. Compile and run!
